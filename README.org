This script will let you easily generate Arch Linux PKGBUILDs for R
packages available on CRAN.

* Requirements

- Ruby
- =nokogiri=

* Usage

Simply run =pacr.rb cran_pkg= (e.g., =pacr.rb ggplot2=) to write an
Arch Linux PKGBUILD to stdout.
